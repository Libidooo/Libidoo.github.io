### 主体优先级

主体判定顺序如下（基于宗派信号可信度排序）：
1. 弥勒（卷十）——信号最强：与唯识宗教义祖师关系，玄奘、窥基系统弘扬
2. 阿弥陀（卷三）——净土宗信仰核心，唐宋造像规模最大
3. 观音（卷二）——已有子分类机制（天台系/心经系/密教系/民间大乘）
4. 地藏（卷九）——无专属宗派，跨宗派存在
5. 药师（卷四）——宗派专属性最弱：横跨净土、密教、天台

主体一旦判定：不得再跨卷判断宗派。
（注：密教W≥60时通过显式逻辑优先，不参与主体优先级排序）

**示例**：出现"药师佛 + 阿弥陀咒文" → S得分接近时主体=阿弥陀 → 判定仍在卷三。

### 交叉引证规则

#### 卷一（通用经文）与其他卷的关系
卷一 = 通用，不得用于宗派判定。
- 若"经文词干"出现 → 可以加强语义，但不能决定宗派
- 若经文词干与卷二/三/四/五的关键字同时出现 → 只作为辅助语义

卷一只参与"增强"，不参与"决定"。

#### 三大混像排除（卷二、卷三、卷四之间）
| 情况 | 判定 |
|------|------|
| 半跏思惟 | 卷十（弥勒） |
| 持锡杖 | 卷九（地藏） |
| 施无畏印 + 甘露瓶 | 卷二（观音） |

#### 思想宗派三角区（卷七、卷十一、卷十二）
| 词语 | 判定 |
|------|------|
| 本来面目 | 禅宗（卷七） |
| 非有非无 / 八不中道 | 三论（卷十一） |
| 八识 / 阿赖耶识 | 唯识（卷十二） |

这三个卷必须保持绝对分离，不得出现交叉词。

#### 密教（卷五）与所有卷的分界
出现以下任意词 → 必须进入卷五：
- 種子字（凡梵字）
- 手印（智拳印等）
- 曼荼罗
- 大日如来
- 不动明王等明王体系

密教是最强"排他卷"。只要密教出现 → 其他卷全部靠后。

#### 区域卷（卷八）逻辑位置
卷八（四川地域词）永远只做"地点辅助"，不得判宗派。

**示例**："资州 + 大悲观音" → 仍判为卷二。

### 权重冲突处理

#### 权重优先级（从高到低）
| 优先级 | 卷别 | 原因 |
|--------|------|------|
| 1 | 卷五（密教） | 本尊 & 種子字最高 |
| 2 | 卷四（药师） | 药器、十二愿极强 |
| 3 | 卷十（弥勒） | 半跏思惟不可被取代 |
| 4 | 卷九（地藏） | 六道、锡杖唯一指向 |
| 5 | 卷二（观音） | 手印体系强但非排他 |
| 6 | 卷三（净土） | 往生文具备排他性 |
| 7 | 卷六（华严） | 有文本强度但图像弱 |
| 8 | 卷七-十一-十二（思想宗派） | 文献型判定，不基于图像 |
| 9 | 卷一（通用） | 不用于决定宗派 |
| 10 | 卷八（区域） | 不决定宗派 |

#### 交叉佐证模板（AI 用）
当两个卷冲突时：
- IF 卷五词出现 → 绝对密教（卷五）
- ELSE IF 主体词（弥勒 / 药师 / 地藏 / 阿弥陀 / 观音） → 按主体卷为主
- ELSE IF 净土文出现 + 无药师/弥勒/地藏 → 卷三
- ELSE IF 空宗逻辑（非有非无） → 卷十一
- ELSE IF 八识（阿赖耶） → 卷十二
- ELSE 经文通用（卷一） → 不参与宗派判定

## 附录：跨卷校对矩阵

下面将三者全部逐卷拆开，以"主体 > 场景 > 文句 > 图像"的方式校准，使未来自动判定体系不会出错。

### 1. 跨卷交叉校对（延寿 / 宝珠 / 施食）

#### Ⅰ. 「延寿」跨卷校对

延寿 / 延命 / 长寿 / 延生是四个最常与宗派混淆的词。必须区分卷四（药师）与卷九（地藏）。

##### Ⅰ–1 延寿词汇在卷四（药师体系）→ 主归属

药师十二愿中最明确的愿：
- "延寿无量愿"
- "除病延寿"
- "增益寿命"
- "除绝命难"

若题记出现以下任一，即强归卷四药师：

| 词条 | 权重 | 原因 |
|------|------|------|
| 延寿 | 75 | 药师十二愿核心 |
| 延生 | 75 | 药师主用词 |
| 延命 | 80 | 药师本愿明确记载 |
| 增寿 | 70 | 药师的"增益寿命"之愿 |
| 消灾延寿 | 80 | 非地藏语，独属药师与大悲系 |

**图像确认点（药师系）**：
- 宝珠在左手
- 右手施无畏（但不同于观音）
- 藏蓝色/深蓝色佛身
- 配十二神将
- 持药钵

地藏没有"药钵 / 药壶 / 十二神将" → 可直接排除卷九。

##### Ⅰ–2 延寿词汇在卷九（地藏体系）→ 辅助因缘

地藏确实也有实践层面的"祈求延寿"题记，如：
- "愿现世父母增福延寿"（卷九 K12）
- "愿离苦得寿安"

但关键区别在于：
地藏题记中，"延寿"永远是 从亡灵拔苦 → 回向给现世父母 的配套句式。

**典型结构**：
"为亡××荐拔，愿在世父母增福延寿。"

如果是这个结构 → 主判卷九（地藏），延寿只是附加句式。

##### Ⅰ–3 延寿判定总规则（最关键）

| 场景特征 | 归属 |
|----------|------|
| 延寿 + 药师名字 | 卷四（药师）主 |
| 延寿 + 十二神将 | 卷四（药师绝对） |
| 延寿 + 亡父母 + 荐拔 | 卷九（地藏）主 |
| 单词"延寿"但主体不明 | 中性，不判宗派 |
| 延寿 + 极乐净土 | 卷三（净土）辅助，但不多见 |
| 延寿 + 放焰口 / 施食 | 卷九（地藏）倾向 |

---

#### Ⅱ. 「宝珠」跨卷校对（最容易误判的关键点）

宝珠（如意宝珠）是佛教最常见法器之一，但不同宗派意义完全不同。故必须按卷别精细区分。

##### Ⅱ–1 宝珠在卷二（观音）→ "如意宝珠"

- 大悲观音常持宝珠
- 四臂观音"主宝珠"是定物（密观音的前身）
- 如意轮观音（密教化）宝轮 + 宝珠

**图像要点**：
- 宝珠发光
- 放于掌中
- 与净瓶、杨柳枝等组合

倾向卷二（观音）。但不是绝对，必须看主体。

##### Ⅱ–2 宝珠在卷三（净土）→ "阿弥陀 / 势至 的宝光珠"

较弱，但存在：
- 大势至菩萨（净土三圣之一）常持宝珠（智慧光明的象征）
- 阿弥陀佛的宝光珠一般在背景装饰

**判定逻辑**：
若出现：
- 阿弥陀佛
- 一佛二菩萨
- 莲池
- 接引

→ 宝珠自动归入 卷三净土，不是观音。

##### Ⅱ–3 宝珠在卷四（药师）→ "药珠 / 医疗光珠"

药师佛常持宝珠，但大多数时候为：
- 药壶 / 药钵
- 药丸（宝珠形）

必须区分：

| 药师宝珠特征 | 普通宝珠区别 |
|--------------|--------------|
| 较大，偏圆，不发光 | 观音宝珠常发光 |
| 与药钵连用 | 観音不持钵 |
| 配十二神将 | 其他卷无 |

##### Ⅱ–4 宝珠在卷九（地藏）→ "明珠（光珠）"

地藏的宝珠被明确记为：
- 明珠（照破地狱）
- 珠光大亮（地藏经描述）

**图像区分方式**：

| 特征 | 地藏 | 观音 |
|------|------|------|
| 珠光更大、光芒外射 | 是 | 否（多为柔光） |
| 常与锡杖并列 | 是 | 否 |
| 站或坐于地狱场景 | 是 | 否 |

若宝珠 + 锡杖 → 必为地藏（卷九）

否则必须按主体判定，而不是按宝珠本身判定宗派。

---

#### Ⅲ. 「施食」跨卷校对（四川特色）

施食、水陆、焰口，是四川石刻中频繁出现的概念。必须明确它们的宗派归属。

##### Ⅲ–1 施食在卷九（地藏）→ 主归属

最典型的施食系句式：
- 普施饿鬼
- 普济幽魂
- 普施冥界众生
- 施食会
- 超度亡灵

若与地藏主体并存 → 卷九主。

##### Ⅲ–2 施食在卷三（净土）→ 次要归属

净土系的施食一般用于：
- 超度宗亲 → 往生极乐
- 水陆法会

**特征**：
- 有"西方""往生""莲花""三圣"
- 没有地狱场景

**作用**：
卷三（净土）把施食看作"荐亡前行"，不是主体。

##### Ⅲ–3 施食在卷五（密教）→ 特殊情况

密教中有：
- 施食法（施食于饿鬼）
- 部分陀罗尼配合施食（如《苏悉地》仪轨）

但： 若施食配合 梵字 / 密印 / 明王 → 必入卷五。

##### Ⅲ–4 施食的判定总规则

| 场景 / 主体 | 归属 |
|-------------|------|
| 地藏 + 施食 | 卷九主 |
| 阿弥陀佛 + 施食 | 卷三主 |
| 密印 + 施食 | 卷五主 |
| 无主体（仅施食） | 中性灭罪仪轨，不归宗派 |
| 孤魂 / 饿鬼 / 冥府 / 普施 | 卷九加权更多 |

### 2. 前十二卷交叉逻辑验证体

这是一个三层结构：
- 层级一：十二卷的逻辑定位（互不跨界）
- 层级二：交叉验证（任意两卷之间的互斥 / 互补）
- 层级三：交叉佐证（显性 + 隐性判定机制）

#### 层级一：十二卷逻辑定位（不可跨界）

以下是十二卷的 唯一且不可被改变的逻辑边界：

| 卷号 | 内容领域 | 本质属性 | 是否主体判定 | 是否宗派判定 |
|------|----------|----------|--------------|--------------|
| 1 | 大乘通用经文 | 通用语汇库 | ❌ | ❌ |
| 2 | 观音体系 | 主体卷 | ✔ | ✔（天台/净土/密） |
| 3 | 净土体系 | 宗派卷 | ✔（阿弥陀） | ✔ |
| 4 | 药师体系 | 主体卷 | ✔（药师） | ✔ |
| 5 | 密教体系 | 宗派卷 | ✔（密本尊） | ✔ |
| 6 | 华严体系 | 宗派卷 | ✔（普贤/毘卢） | ✔ |
| 7 | 禅宗体系 | 宗派卷（思想卷） | ❌ | ✔（禅宗） |
| 8 | 区域体系 | 辅助卷 | ❌ | ❌ |
| 9 | 地藏体系 | 主体卷 | ✔（地藏） | ❌（非宗派） |
| 10 | 弥勒体系 | 主体卷 | ✔（弥勒） | ❌（非宗派） |
| 11 | 三论宗 | 宗派卷（思想卷） | ❌ | ✔ |
| 12 | 唯识宗 | 宗派卷（思想卷） | ❌ | ✔ |

四个主体卷（2/3/4/9/10）必须先判主体，再判宗派。非主体卷不得擅自判主体。

#### 层级二：十二卷交叉逻辑验证（核心部分）

交叉验证 = 避免卷之间产生矛盾 / 混淆 / 误归属。

##### 2.1 卷一（通用经文）与其他卷的关系

卷一 = 通用，不得用于宗派判定。
- 若"经文词干"出现 → 可以加强语义，但不能决定宗派
- 若经文词干与卷二/三/四/五的关键字同时出现 → 只作为辅助语义

卷一只参与"增强"，不参与"决定"。

##### 2.2 主体判定优先序（五大主体）

主体识别顺序如下（不可更改）：
1. 药师（卷四）
2. 弥勒（卷十）
3. 地藏（卷九）
4. 观音（卷二）
5. 阿弥陀（卷三）

主体一旦判定：不得再跨卷判断宗派。

**示例**：出现"药师佛 + 阿弥陀咒文" → 主体=药师 → 判定仍在卷四。

##### 2.3 三大混像排除（卷二、卷三、卷四之间）

观音 vs 弥勒 vs 地藏

| 情况 | 判定 |
|------|------|
| 半跏思惟 | 卷十（弥勒） |
| 持锡杖 | 卷九（地藏） |
| 施无畏印 + 甘露瓶 | 卷二（观音） |

##### 2.4 思想宗派三角区（卷七、卷十一、卷十二）

禅宗（卷七） vs 三论宗（卷十一） vs 唯识宗（卷十二）

| 词语 | 判定 |
|------|------|
| 本来面目 | 禅宗（卷七） |
| 非有非无 / 八不中道 | 三论（卷十一） |
| 八识 / 阿赖耶识 | 唯识（卷十二） |

这三个卷必须保持绝对分离，不得出现交叉词。

##### 2.5 密教（卷五）与所有卷的分界

出现以下任意词 → 必须进入卷五：
- 種子字（凡梵字）
- 手印（智拳印等）
- 曼荼罗
- 大日如来
- 不动明王等明王体系

密教是最强"排他卷"。只要密教出现 → 其他卷全部靠后。

##### 2.6 区域卷（卷八）逻辑位置

卷八（四川地域词）永远只做"地点辅助"，不得判宗派。

**示例**："资州 + 大悲观音" → 仍判为卷二。

### 3. 新增11-15卷卷内交叉逻辑（操作细则）

本附录用于将互斥矩阵（⛔/⚠）与模板C冲突处理，落到可执行的"记分、退场、输出"步骤中；适用于所有宗派卷，重点约束卷11–卷15。

#### 一、基本定义（不改变原规则）

1）主判卷：在完成主体优先与互斥矩阵检查后，S 值最高且满足阈值的卷。
2）背景卷：与主判卷标记为⚠允许并存，但未满足阈值或被主判卷压制的卷；只能进入"备注/背景交叉引证"，不得写入宗派栏。
3）退场卷：未满足 Gate 或被⛔互斥排除的卷，W=0，只能记录为"出现过但不计分（原因：××）"。

（以上用于满足"透明性要求：必须输出W→S过程与证据链"）

---

#### 二、跨卷并存时的记分规则（⚠并存专用）

1）当两卷为 ⚠（允许并存）时：
- 仍按各自词表计算 S
- 不得把A卷词条加到B卷里"凑分"
- 输出时只能有一个"宗派栏主判卷"，其他卷只能写入"备注：背景卷证据"

2）当某卷仅由"弱因缘/高泛化词"构成（例如卷11的"空/有/假名"、卷15的"持戒/净戒"、卷13的"法华经经名"等）：
- 即便这些词在文本中出现频繁，仍不得触发 Gate；该卷直接退场或仅作背景。

（该条承接你总纲"不得以经验替代判定词""不得从经名/品名反推宗派"的禁则。）

---

#### 三、卷11–卷15的"互相冲突消解规则"（新增卷的关键）

以下为操作级"优先消解"条款（不替代互斥矩阵，仅用于⚠并存时的退场判定）：

##### 1）卷12（唯识）优先消解
若出现卷12绝对锚点（如"八识/阿赖耶/三性"等类型锚点），卷11/13/14退为背景；不得用"空/中道/实相"等把结论拉回三论或天台。

**逻辑来源**：思想卷需硬锚点，泛词不得抢判。

##### 2）卷11（三论）消解
若仅出现"空、有、假名"等高泛化词 → 卷11不入判定（退场或背景），除非同时出现卷11硬锚点（论名/祖师/破遣结构等）。

##### 3）卷13（天台宗）消解
若主体=观音且已触发卷二"天台系观音"决定因 → 宗派优先写卷二分支；卷13仅作背景引证。

**保证**："卷二内部路径"不被"独立天台卷"重复覆盖。

##### 4）卷14（成实宗）消解
若未出现"成实论/作者/五聚/关键原句/义疏题名"等论典硬锚点 → 卷14不得成立；不得以"四谛/空/二谛"等通用义理词推定成实。

**承接**："思想史判定卷须慎用"的总纲警告。

##### 5）卷15（律宗）消解
若未出现"律典/戒本/羯磨程序/受具足戒制度/道宣南山著述"等硬锚点 → 卷15不得成立；不得以"持戒清净"等德目赞叹语判律宗。

#### 层级三：交叉佐证（跨卷权重整合）

为了自动化系统，需建立"交叉权重体系"：

##### 3.1 权重优先级（从高到低）

| 优先级 | 卷别 | 原因 |
|--------|------|------|
| 1 | 卷五（密教） | 本尊 & 種子字最高 |
| 2 | 卷四（药师） | 药器、十二愿极强 |
| 3 | 卷十（弥勒） | 半跏思惟不可被取代 |
| 4 | 卷九（地藏） | 六道、锡杖唯一指向 |
| 5 | 卷二（观音） | 手印体系强但非排他 |
| 6 | 卷三（净土） | 往生文具备排他性 |
| 7 | 卷六（华严） | 有文本强度但图像弱 |
| 8 | 卷七-十一-十二（思想宗派） | 文献型判定，不基于图像 |
| 9 | 卷一（通用） | 不用于决定宗派 |
| 10 | 卷八（区域） | 不决定宗派 |

##### 3.2 交叉佐证模板（AI 用）

当两个卷冲突时：
- IF 卷五词出现 → 绝对密教（卷五）
- ELSE IF 主体词（弥勒 / 药师 / 地藏 / 阿弥陀 / 观音） → 按主体卷为主
- ELSE IF 净土文出现 + 无药师/弥勒/地藏 → 卷三
- ELSE IF 空宗逻辑（非有非无） → 卷十一
- ELSE IF 八识（阿赖耶） → 卷十二
- ELSE 经文通用（卷一） → 不参与宗派判定
