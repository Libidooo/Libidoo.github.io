## 概览
卷八为区域体系，不是主体卷、也不是宗派卷。它提供区域、匠人、地名、时代、方言式词根，完全作为辅助性语义，不参与宗派判定。

## 核心内容

### 卷八总说明

**卷八功能定位**：
- 卷八不是主体卷、也不是宗派卷
- 提供区域、匠人、地名、时代、方言式词根
- 完全作为辅助性语义，不参与宗派判定

**判定对象**：
- 地名（安岳、简州、大足）
- 匠人名号
- 行政区、寺院名
- 年号（用于判断佛教流派时期盛衰）

**触发条件**：
- 出现地名 / 州名
- 匠人款识
- 特殊地区用语（如四川方言式用字）
- 年号（如开成、咸通）

**强因缘 / 弱因缘 / 排除项**：
- **强因缘（区域背景）**：大足、安岳等关键石窟地名（W=60-95）
- **弱因缘**：寺院名、州名（W=60-80）
- **排除项**：
  - 匠人名绝不可作为宗派判断理由
  - 州名不可引导"宗派倾向"

**卷间交叉逻辑**：
- 卷八从不决定宗派
- 仅加强：卷二（四川观音）、卷五（四川密教）、卷九（酆都信仰）
- 绝不可覆盖卷五、卷三等主体卷的输入

### V8-1：区域体系（R-Series）

区域体系帮助自动分类器判断区域 → 宗派的可能性倾向。区域词条不决定宗派，但会影响"权重加成"（+5至+25分）。

#### R1：四川核心区域（权重 60–90）

| 区域 | 权重W | 宗派倾向 | 说明 |
|------|-------|----------|------|
| 大足 | 90 | 华严 + 密教（并存） | 大足是全国最复杂的宗派交会地 |
| 安岳 | 85 | 华严（最稳定） | 文殊普贤、毘卢遮那极多 |
| 荣县 | 70 | 华严 + 净土 | 荣县大佛典型显教佛 |
| 简州 | 75 | 华严 + 佛会图 | 多层楼阁的大型佛会图 |
| 蓬州 | 60 | 地方性小型华严 | 善财、文殊普贤出现多 |
| 成都（大慈寺） | 80 | 密教强 | 唐代密教重镇 |
| 青城山 | 55 | 禅宗弱 + 道佛混合 | 需特别排除道教符号 |

#### R2：关中（长安）区域（W=70–95）

| 区域 | 权重 | 宗派倾向 |
|------|------|----------|
| 长安 | 95 | 密教（最强） |
| 扶风 / 宝鸡 | 80 | 法华 + 天台 |
| 西安 | 75 | 多宗共存 |

#### R3：江浙区域（W=70–85）

| 区域 | 权重 | 倾向 |
|------|------|------|

| 台州 | 80 | 天台 |
| 杭州 | 75 | 禅宗 + 净土 |
| 宁波 | 70 | 法华 |

#### R4：江南区域（W=60–80）

| 区域 | 权重 | 倾向 |
|------|------|------|
| 苏州 | 60 | 禅宗 |
| 扬州 | 70 | 净土 + 华严 |
| 南京 | 65 | 禅 + 净土 |

#### R5：河北、河南（W=70–90）

| 区域 | 权重 | 倾向 |
|------|------|------|
| 嵩山 | 90 | 禅宗（达摩、少林） |
| 洛阳 | 75 | 净土 + 法华 |

**区域体系总结**：
- 区域并不直接判定宗派，但提供倾向性加成（+5至+25分）
- 四川体系将特别用于卷六、卷五、卷七的交叉判断

### V8-2：年号体系（N-Series）

年号 = 佛教流派传播节奏的最关键历史信息之一。年号将作为"宗派倾向加成"（+10–30分）。

#### N1：隋代（开皇 581–600 / 大业 605–617）

| 年号 | 倾向 |
|------|------|
| 开皇 | 天台宗强（智顗） |
| 仁寿 | 天台 |
| 大业 | 天台 + 法华 |

#### N2：唐代前期（贞观–开元）

| 年号 | 倾向 | 说明 |
|------|------|------|
| 贞观 | 法相宗（唯识） | 玄奘时代 |
| 永徽 | 唯识 | |
| 咸亨 | 三论 | |
| 上元 | 密教开始入华 | |

#### N3：唐代中期（开元–天宝）密教最强

| 年号 | 倾向 |
|------|------|
| 开元 | 密教绝对强势（善无畏、金刚智） |
| 天宝 | 密教 |
| 宝应 | 密教 |

**判定规则**：任何"开元、天宝"年份 → 图像与题记极可能密教化。

#### N4：唐代晚期（大历–咸通）华严强

| 年号 | 倾向 |
|------|------|
| 大历 | 华严 + 密 |
| 建中 | 华严 |
| 元和 | 华严 |
| 会昌 | 佛教灭法（需谨慎判定） |
| 咸通 | 华严极强（大足核心时期） |

#### N5：五代十国（前蜀）

| 年号 | 倾向 |
|------|------|
| 天复 / 光化 / 广明 | 四川密教 + 华严并存 |
| 前蜀天成 | 华严强 |

#### N6：宋代（太平兴国–绍兴）

| 年号 | 倾向 |
|------|------|
| 太平兴国 | 禅宗强 |
| 天禧 | 禅 + 净土 |
| 熙宁 / 元丰 | 宋密（弱） |
| 绍兴 | 禅宗最盛 |

**年号体系总结**：
- 年号可以给宗派加成（+10–30分）
- 若题记出现"咸通、开元、天宝" → 优先考虑华严/密教
- 若题记出现"开皇、大业" → 天台加权
- 若出现"太平兴国、绍兴" → 禅宗加权

### V8-3：匠人体系（C-Series）

在四川，匠人具有极高的流派偏向。匠人名号提供 +5–20 加权。

#### C1：四川匠人（核心示例）

| 匠人名 | 区域 | 倾向 | 说明 |
|--------|------|------|------|
| 陈君贵 | 安岳 | 华严倾向极强 | |
| 刘刻师 | 安岳 | 华严 | |
| 陈用之 | 大足 | 华严 + 密 | |
| 龙昌 | 大足 | 密教倾向 | |
| 陈文德 | 荣县 | 净土 | |
| 王大昌 | 简州 | 佛会图 | |

#### C2：匠人判定规则

- 匠人不是决定性判定，但可提供 +5–20 加权
- 若题记中匠人名号同"安岳匠人"群符合 → 优先华严
- 若题记出现在大足 + 匠人"龙昌" → 必须考虑密教组合图像
- 荣县匠人常配合阿弥陀佛 → 净土偏向

### V8-4：异体字体系（E-Series）

题记中有大量异体字，若不识别OCR会误判。此体系不决定宗派，但为"识别正确性"提供重要基础。

**核心异体字类别**：
- 佛号类
- 菩萨名号类
- 宗派关键词类
- 地理名称类

**说明**：
- 异体字体系不决定宗派
- 为OCR识别和语义理解提供基础
- 详细异体字表见references/detail/volume-eight-regional-full.md

## 验证规则

### 判定检查清单
- [ ] 出现区域地名（四川、长安、江浙等）
- [ ] 出现匠人名号
- [ ] 出现年号（开皇、开元、咸通等）
- [ ] 权重仅用于倾向性加成，不决定宗派

### 跨卷冲突处理
- **卷八从不决定宗派**
- 仅加强：卷二（四川观音）、卷五（四川密教）、卷六（四川华严）、卷九（酆都信仰）
- 绝不可覆盖卷五、卷三等主体卷的输入

### 常见错误
- ❌ 错误：将区域地名直接判定为宗派
- ❌ 错误：将匠人名号作为宗派判断理由
- ❌ 错误：忽略年号的倾向性加成作用

### 正确示例
```
✅ 正确：
题记："咸通年，安岳造毗卢遮那佛"
主体判定：毗卢遮那佛（华严）
宗派判定：咸通（华严加成+30）+ 安岳（华严加成+25）→ 华严体系（卷六）

✅ 正确：
题记："开元年，长安造大日如来"
主体判定：大日如来（密教）
宗派判定：开元（密教加成+30）+ 长安（密教加成+25）→ 密教体系（卷五）

❌ 错误：
题记："安岳造佛像"
主体判定：佛像
宗派判定：？？？
（错误：安岳是区域，不能直接判定为华严，必须有其他华严特征）

❌ 错误：
题记："匠人陈君贵造"
主体判定：无明确主体
宗派判定：？？？
（错误：匠人名号不能作为宗派判断理由）
```

## 约束与注意事项
- 卷八不是主体卷、也不是宗派卷
- 完全作为辅助性语义，不参与宗派判定
- 区域词条提供倾向性加成（+5至+25分）
- 年号提供倾向性加成（+10至+30分）
- 匠人名号提供倾向性加成（+5至+20分）
- 匠人名绝不可作为宗派判断理由
- 州名不可引导"宗派倾向"
- 卷八从不决定宗派
