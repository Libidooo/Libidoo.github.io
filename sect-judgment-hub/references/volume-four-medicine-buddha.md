## 概览
卷四主体必须为药师佛（药师如来 / 药师琉璃光如来），不推断其他宗派。药师十二大愿是宗派判定的核心决定因，一旦出现必须判定为药师体系。

## 核心内容

### 卷四总说明

**卷四顶层规则**：
- 主体必须为药师佛（药师如来 / 药师琉璃光如来），不推断其他宗派
- 药师十二大愿 → 宗派判定核心决定因，一旦出现必须判定为药师体系
- 药师佛名号、琉璃、七佛、十二神将 → 强决定因
- 药师专属咒语词干 → 强因缘
- 所有非药师词条必须按规则迁往其他卷（如菩萨名号、地狱、极乐、观音名称等）
- 图像学仅作辅助，不可决定宗派——但"药师三尊（药师 + 日光 + 月光）"可以作为强因缘
- 每条必须具有文献来源（来自《药师琉璃光本愿功德经》）

**权重体系**：
- W 80–75 → 决定因
- W 70–55 → 强因缘
- W 45–35 → 增上缘
- W 20–30 → 弱增上缘
- W 0 → 異體字
- W < 0 → 排除并迁移

## 验证规则

### 判定检查清单
- [ ] 主体已确定为药师佛（药师如来/药师琉璃光如来）
- [ ] 出现药师十二大愿决定因（W≥75）或药师名号（W≥75）
- [ ] 出现东方琉璃世界词干（W≥75）或药师七佛（W≥70）
- [ ] 出现十二神将（W≥70）或日光月光菩萨（W≥70）
- [ ] 无负向排除因素（净土、地藏、弥勒、观音、密教、华严、唯识、三论专属词）
- [ ] 权重总和S ≥ 100
- [ ] 文献来源明确

### 跨卷冲突处理
- 与卷三（净土）：出现"药师 + 往生" → 主体优先药师（卷四），标记为"药师净土混合信仰"
- 与卷五（密教）：出现"药师 + 真言/陀罗尼" → 卷四与卷五并列，标记为"药师 + 密教化"
- 与卷二（观音）：出现"药师 + 观音" → 主体优先药师，观音仅作助侍
- 与卷九（地藏）：出现"药师 + 拔苦" → 主体优先药师，地藏词作为功能语

### 常见错误
- ❌ 错误：将所有"有药师"题记一律归入净土
- ❌ 错误：将"琉璃光"判为净土（琉璃光是药师专属）
- ❌ 错误：忽略"十二大愿"的决定性作用
- ❌ 错误：将"东方琉璃世界"判为净土
- ❌ 错误：忽略负向排除规则

### 正确示例
```
✅ 正确：
```

## 约束与注意事项
- 卷四主体必须为药师佛，不推断其他宗派
- 药师十二大愿是核心决定因（W≥75），一旦出现必须判定为药师体系
- 药师佛名号、东方琉璃世界、药师七佛、十二神将均为强决定因
- 东方琉璃世界词干优先级高于卷三净土的极乐世界
- 出现"药师 + 往生" → 主体优先药师（卷四），标记为"药师净土混合信仰"
- 12类负向排除规则必须严格执行
- 权重严格按照80/75/70/55/45/35/20–30分级
- 跨卷冲突处理：药师优先、与净土混合、与密教化并列
