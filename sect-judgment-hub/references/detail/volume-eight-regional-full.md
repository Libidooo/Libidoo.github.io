# 卷八区域体系（完整版）

卷八
区域体系 / 年号体系 / 匠人体系 / 异体字体系）
（1）本卷功能定位
卷八不是主体卷、也不是宗派卷。它提供：区域、匠人、地名、时代、方言式词根完全作为 辅助性语义，不参与宗派判定。
（2）判定对象
地名（安岳、简州、大足）
匠人名号
行政区、寺院名
年号（用于判断佛教流派时期盛衰）
（3）触发条件
出现地名 / 州名
匠人款识
特殊地区用语（如四川方言式用字）
年号（如开成、咸通）
（4）强因缘 / 弱因缘 / 排除项
（注意：这里权重不用于宗派，只用于强化背景）
强因缘（区域背景）：大足、安岳等关键石窟地名
弱因缘：寺院名、州名
排除项：
匠人名绝不可作为宗派判断理由
州名不可引导“宗派倾向”
（5）卷间交叉逻辑
卷八从不决定宗派
仅加强：卷二（四川观音）、卷五（四川密教）、卷九（酆都信仰）
绝不可覆盖卷五、卷三等主体卷的输入
全卷结构如下：

（四川 + 全国可迁移区域）
目标：帮助自动分类器判断 区域 → 宗派的可能性倾向。例如：
四川 → 华严 + 密教倾向高
长安 → 密教倾向极高
江浙 → 天台、法华倾向高
河南 → 禅宗倾向高
区域词条不决定宗派，但会影响“权重加成”。

区域体系总结
区域并不直接判定宗派，但提供 倾向性加成（+5 至 +25 分）。四川体系将特别用于卷六、卷五、卷七的交叉判断。
V8-2：年号体系（N-Series）
年号 = 佛教流派传播节奏的最关键历史信息之一。
例如：
开元 / 天宝 → 密教大盛
咸通 → 华严巨大扩展
开皇 / 大业 → 天台强势期
年号将作为“宗派倾向加成”。

年号体系总结
年号可以给宗派加成（+10–30 分）：
若题记出现“咸通、开元、天宝” → 优先考虑华严/密教
若题记出现“开皇、大业” → 天台加权
若出现“太平兴国、绍兴” → 禅宗加权
V8-3：匠人体系（C-Series）
在四川，匠人具有极高的流派偏向。例：
安岳匠人 → 华严倾向
大足匠人 → 华严 + 密教
简州匠人 → 佛会图
荣县匠人 → 净土
因此必须收录匠人名号。
