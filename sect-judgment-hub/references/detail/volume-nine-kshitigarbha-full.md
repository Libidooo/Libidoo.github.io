# 卷九地藏体系（完整版）

## 第一部分：卷九总则

卷九总则（地藏体系）
适用范围
专门处理地藏菩萨、地藏王、十王、地狱图、荐亡 / 超度等全部“地藏信仰”相关要素。
主体与宗派关系
主体可为单尊地藏、地藏与十王、地藏与地狱众生等。
出现“地藏名号 + 地狱 / 十王 / 开天堂路闭地狱门等语” → 直接确立地藏体系。
权重使用原则
《地藏经》《地藏十轮经》等经名 + 地藏名号 → 宗派决定因（70–80）。
“荐亡、拔苦、超度、冥报”等若明确与地藏相连 → 视为 强因缘（45–55）。
交叉引证方式
与卷三：若同时出现“往生极乐、度亡”则为“净土 + 地藏”。
与卷五：若出现地藏真言或密教化十王像，则并列“地藏 + 密教化”。
负向因与使用禁则
禁止把所有“荐亡、拔苦”自动归入地藏 → 必须有地藏/地狱语境。
地狱图若明显与药师/弥勒体系不符，且无地藏，则需保持未决，不强行填补。
特别说明
卷九聚焦“死后世界 + 度亡”的佛教表达，与纯粹“祈福延寿”的药师体系有所不同。

## 第二部分：卷九主内容

卷九 · 地藏体系（Kṣitigarbha Corpus）
本卷功能定位
卷九用于 地藏菩萨主体 与 六道 / 救苦 / 冥府信仰 的判定，不属于某宗派，而是 独立信仰体系卷。
（2）判定对象
主体：地藏菩萨
范畴：六道、亡者救度、冥府场景
（4）强因缘 / 弱因缘 / 排除项
强因缘：六道词、地藏名号、锡杖、冥府词
弱因缘：救苦、拔罪、荐亡
排除项：
往生词根（卷三）
密教五明王词（卷五）
半跏思惟（卷九）
藥師十二大願词（卷四）药师十二大愿词（卷四）
（3）判定触发条件
出现“地藏 / 地臧 / 地蔵”
六道词（地狱、饿鬼、畜生、人道、天、修罗）
冥府词（酆都、阎罗、判官等）
持锡杖、明珠（二法器）
（5）卷间交叉逻辑
若出现“六道 + 往生” → 分割判定
“施食 + 六道” → 卷九
