# 卷四药师体系（完整版）

卷四：药师佛体系
卷四将严格遵守以下原则：
卷四顶层规则：
主体必须为药师佛（药师如来 / 药师琉璃光如来）不推断其他宗派。
药师十二大愿 → 宗派判定核心决定因一旦出现，必须判定为药师体系。
药师佛名号、琉璃、七佛、十二神将 → 强决定因
药师专属咒语词干 → 强因缘
所有非药师词条必须按规则迁往其他卷（如菩萨名号、地狱、极乐、观音名称等）
图像学仅作辅助，不可决定宗派— 但“药师三尊（药师 + 日光 + 月光）”可以作为强因缘。
每条必须具有文献来源（来自《药师琉璃光本愿功德经》）
权重＝宗派信任度依据· W80–75 → 决定因· W70–55 → 强因缘· W45–35 → 增上缘· W20–30 → 弱增上缘· W0 → 異體字· W<0 → 排除并迁移
