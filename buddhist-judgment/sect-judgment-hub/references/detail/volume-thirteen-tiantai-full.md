# 卷十三：天台宗体系

卷十三 天台宗
L13-0 卷十三总则（V10.1-Add）
0.1 适用范围
本卷用于判定题记、碑记、经幢、塔铭、造像记中出现的**天台宗（教观体系）归属。判定以"可核验的文献锚点（书名/章疏名/固定教观公式）"为核心，禁止以泛大乘语汇（如"实相、中道、圆教、妙法"等）直接归宗。
0.2 与既有卷的边界（避免重复判定）
你原体系已经在**卷二（观音体系）**内置"天台宗（法华系）观音体系"，且该章已逐条给出"词条—文献来源—W—T"。因此本卷十三为"非限定观音主体"的天台宗派卷：
若主体=观音，且出现卷二"普门/普门示现/三十二应/海潮音"等决定因，则宗派优先按卷二输出"天台系（观音内部分支）"，本卷仅可作为"背景交叉引证"。
若主体≠观音或题记为"教观/讲习/章疏/判教"性质，则可进入本卷计分。
0.3 总纲硬规则继承（本卷不得改写）
主体不明 → 不得判宗派。
宗派判定必须输出 W→S 计算过程、区分主要/次要证据。
次要证据（地点、时代趋势等）必须减半计入且不得压过主要证据。
密教卷例外（卷五）：出现任一 W≥60 密教词 → 必入卷五。
L13-1 进入门槛（Gate：无 Gate 不得计分）
为杜绝"见法华语就判天台"的误判，本卷必须满足以下任一 Gate 才能进入计分；否则本卷退场（W=0，仅可备注）：
Gate-A（书名锚点）：出现本卷 L13-2.1 所列任一"天台核心典籍/章疏名"（如《摩诃止观》《法华玄义》《天台四教仪》《四教义》《止观义例》《十不二门》《教观纲宗》等）。
Gate-B（宗名锚点）：出现"天台宗/天台教门/天台教观"等宗派自称（但仍需排除地名/寺名误触发，见禁则）。
Gate-C（教观公式锚点）：出现"五时八教""一心三观""一念三千""十界互具/十界性融互具""藏通别圆（四教）"等固定公式中任一，并至少再出现一个"天台祖师/章疏/书名"辅助锚点。
L13-2 权重体系（W=20–85，按总纲分段）
权重分段与 S 阈值完全沿用总纲：强证据 60–79；中证据 40–59；弱证据 20–39；<20 无判定力。

L13-3 主要证据 / 次要证据分类（强制标注）
本卷 L13-2.1（书名）、L13-2.2（固定公式）、L13-2.3（祖师+语境锚点）在铭文中出现 → 主要证据 1:1 计入。
地点（如"天台山/国清寺"）、时代趋势、施主身份、一般构图等 → 次要证据减半计入，且不得反压主要证据。
L13-4 负向禁则（严格执行）
仅见经名（法华经等）→ 不得判天台；只可作为"卷一通用/法华影响背景"。
仅见抽象词（实相、中道、圆、妙法、真如）→ 一律不进入 Gate-C；不得锁宗派。
"天台"若仅为地名/寺名构件（如"天台山某寺"）→ 默认次要证据减半，且不得单独触发本卷 Gate。
若出现密教强因 W≥60 → 必入卷五，本卷退位（总纲硬规则）。
L13-5 交叉冲突处理（按总纲模板 C）
若卷二观音已成立（主体=观音）且出现"普门/三十二应/海潮音"等 → 宗派按卷二"天台系观音"，卷十三仅做交叉引证。
若出现"八识/阿赖耶"等唯识绝对点 → 依卷十二优先（思想卷互斥按矩阵与模板处理）。
若出现"往生/极乐/九品"等净土强语 → 依卷三优先，不得因"天台地域/寺名"改判。
L13-6 输出要求（透明性）
按总纲规定：输出必须列出全部 W 条目、主要/次要分类、S 结果与等级，才能通过可复查性要求。
