# 卷十一：三论宗体系（Sanlun System）

卷十一的原则（V10.1绑定）
三论宗 = 中国八宗之一（吉藏体系）其判定必须完全依据三部论：
《中论》
《十二门论》
《百论》绝不能凭空创造术语。
卷十一 = 三论宗的"宗派判定卷"不是哲学 summary，而是：题记 / 铭文 / 图像 → 是否对应三论宗？
卷十一必须满足"主体判定优先"主体若非"空宗倾向"，不得强行归入三论宗。
卷十一不能与卷一（般若）重复凡属般若通用词 → 卷一凡属三论宗"特有语气 / 逻辑词" → 卷十一
卷十一不涉及图像（除非用于排除）因三论宗无专属造像。
我将严格遵守以上原则。
