# 卷十：弥勒体系（Maitreya System）

卷十
卷十：弥勒体系（Maitreya System）
（1）本卷功能定位
卷十用于判定 弥勒信仰体系，属于"主体卷"。不承担宗派判定功能，只判定主体是否为弥勒。
（2）判定对象
弥勒上生（兜率天）
弥勒下生（未来佛）
半跏思惟像
（3）判定触发条件（任一即可）
半跏思惟姿（最强判定）
兜率天词根
弥勒名号（异体包括"𠠇勒""弥楽"）
下生经词干（龙华三会）
（4）强因缘 / 弱因缘 / 排除项
强因缘：半跏思惟、兜率天、龙华三会
弱因缘：菩萨名号中的"慈氏"
排除项：
西方三圣（卷三）
锡杖（卷九）
梵字（卷五）
药师十二大愿词（卷四）
（5）卷间交叉逻辑
半跏思惟出现 → 绝对优先卷十
若"思惟 + 莲花台" → 卷十，而非卷三
若"弥勒 + 施食" → 不属于地藏体系（排除卷九）
弥勒不可归于禅宗（卷七）
