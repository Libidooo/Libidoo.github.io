# 卷六华严体系（完整版）

卷六 · 华严体系
卷六属于“显教·华严体系（法界缘起体系）”，不可与密教胎藏界/金刚界混淆。
显教华严 · 毘卢遮那主体判定体系（含图像学规则）**
H1 总原则（必须写入文稿）
毘卢遮那（华严） ≠ 大日如来（密教）两者同样源自梵语 Vairocana，但在汉地佛教的系统结构、语境、经文引用、图像形式全然不同。因此必须通过“主体优先规则 + 语境规则 + 图像规则”三重交叉判定。
卷六 H1 的任务，就是：
避免把显教毘卢误判为密教大日（常见错误）
避免把密教大日误判为华严毘卢（同样危险）
避免“佛光”“普照”等宽泛词误导宗派判定
本节完成 H1 全面校对与图像规则补强。

T4-3 文殊乘狮 / 普贤乘象（并列出现）
此为《华严经》菩萨会核心配置。
权重（W）：75说明：必须为“并列呈现”，单独出现的普通文殊/普贤不可自动判华严。

权重（W）：70说明：必须区分密教胎藏界（呈圆形院落），华严呈“楼阁群落”。
T5：法界缘起图像（W = 75）
华严的最大特征即“法界缘起”结构，典型图像有：

T5-2 “一即一切 / 一切即一”视觉表达

T7：华严“非密教”排除规则（必须加入）
为了避免误判，必须加入以下排除规则：
T7-1 若出现以下手印 → 必定非华严，而是密教：
智拳印
法界定印
三昧耶印
金刚拳印
金刚合掌印
五股杵印
T7-2 若出现曼荼罗结构（院名） → 非华严
包括：
中台八叶院
金刚界三部曼荼罗
外院二十五尊
→ 这些属于卷五。
T7-3 若出现咒语 → 必非华严
尤其是：
唵 / 吽 / 吽佉 / 悉陀
种子字（梵字）
陀罗尼长句
华严体系不使用咒语作为题记关键词。
T8：华严佛像 + 密教风格的混像排查
四川地区常见混像，必须加入严格规则：
T8-1 若佛像为毘卢遮那，但戴密教宝冠 → “混像”
必须按如下规则处理：
若题记有华严词 → 归入华严（优先华严）
若题记有密教词 → 归入密教（优先密教）
若无题记，但图像呈密教印 → 判密教
若无题记，但作华严构图（普贤/文殊对侍） → 判华严
这是防止误判的关键。
T8-2 若毘卢遮那置于多层世界中心 → 归华严
若置于八叶院中心 → 归密教（胎藏界）。
T8-3 若佛像周围有“善财童子 + 普贤 + 文殊”组合 → 强华严（W=80）
T4–T8 小结：
这些图像规则能够：
完全避免华严与密教大日的误判
将四川地区混像正确归类
保证主体优先原则一致性
与卷二（观音）与卷五（密教）无冲突
权重体系完整可用
