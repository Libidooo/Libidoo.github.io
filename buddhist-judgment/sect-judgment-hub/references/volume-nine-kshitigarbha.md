## 概览
卷九用于地藏菩萨主体与六道/救苦/冥府信仰的判定，不属于某宗派，而是独立信仰体系卷。卷九专门处理地藏菩萨、地藏王、十王、地狱图、荐亡/超度等全部"地藏信仰"相关要素。

## 核心内容

### 卷九总说明

**卷九功能定位**：
- 卷九用于地藏菩萨主体与六道/救苦/冥府信仰的判定
- 不属于某宗派，而是独立信仰体系卷
- 专门处理地藏菩萨、地藏王、十王、地狱图、荐亡/超度等全部"地藏信仰"相关要素

**判定对象**：
- 主体：地藏菩萨
- 范畴：六道、亡者救度、冥府场景

**判定触发条件**：
- 出现"地藏/地臧/地蔵"
- 六道词（地狱、饿鬼、畜生、人道、天、修罗）
- 冥府词（酆都、阎罗、判官等）
- 持锡杖、明珠（二法器）

**强因缘 / 弱因缘 / 排除项**：
- **强因缘**：六道词、地藏名号、锡杖、冥府词
- **弱因缘**：救苦、拔罪、荐亡
- **排除项**：
  - 往生词根（卷三）
  - 密教五明王词（卷五）
  - 半跏思惟（卷九）
  - 药师十二大愿词（卷四）

**权重使用原则**：
- 《地藏经》《地藏十轮经》等经名 + 地藏名号 → 宗派决定因（W=70-80）
- "地藏名号 + 地狱/十王/开天堂路闭地狱门等语" → 直接确立地藏体系
- "荐亡、拔苦、超度、冥报"等若明确与地藏相连 → 视为强因缘（W=45-55）

**卷间交叉逻辑**：
- 若出现"六道 + 往生" → 分割判定
- "施食 + 六道" → 卷九
- "施食 + 往生句" → 卷三
- 冥府词绝不去卷五密教（如"冥王"≠明王）
- 与卷三：若同时出现"往生极乐、度亡"则为"净土 + 地藏"
- 与卷五：若出现地藏真言或密教化十王像，则并列"地藏 + 密教化"

**负向因与使用禁则**：
- 禁止把所有"荐亡、拔苦"自动归入地藏 → 必须有地藏/地狱语境
- 地狱图若明显与药师/弥勒体系不符，且无地藏，则需保持未决，不强行填补

**特别说明**：
- 卷九聚焦"死后世界 + 度亡"的佛教表达
- 与纯粹"祈福延寿"的药师体系有所不同

### K1：地藏本愿三卷·文献词干体系（W=30-90）

地藏卷必须以《地藏菩萨本愿经》作为核心依据。整理方式：地名词干、誓愿词干、超度词干、功德词干、地狱词干。

| 词条 | W | 解释 |

| 众生度尽 方证菩提 | 85 | 地藏愿心 |
| 南无地藏王菩萨 | 80 | 题记最高频 |
| 大愿地藏王 | 80 | 地藏专属称号 |

#### K1-B：冥府救度词干（W=60-80）

| 超度 | 70 | 地藏/净土共享 |

### K2：十王体系

十王是地藏体系的重要组成部分，用于冥府审判。

| 词条 | W | 判定倾向 |
|------|---|----------|
| 十王 | 85 | 地藏体系 |
| 阎罗王 | 80 | 地藏体系 |
| 秦广王 | 75 | 十王之一 |
| 楚江王 | 75 | 十王之一 |
| 宋帝王 | 75 | 十王之一 |
| 五官王 | 75 | 十王之一 |
| 阎摩王 | 75 | 十王之一 |
| 变成王 | 75 | 十王之一 |
| 泰山王 | 75 | 十王之一 |
| 平等王 | 75 | 十王之一 |
| 都市王 | 75 | 十王之一 |
| 转轮王 | 75 | 十王之一 |

**判定规则**：
- 若出现"十王" → 判地藏体系（卷九）
- 若出现"地藏十王/酆都/阎罗" → 进入卷九（地藏体系）
- 十王出现 ≠ 必然地藏 → 必须与地藏名号/地狱词共现

### K3：负向排除规则

以下词条出现时，必须排除地藏判定，迁移至其他卷：

| 词条 | 必迁往 | 理由 |
|------|--------|------|
| 阿弥陀佛、往生极乐 | 卷三 · 净土 | 净土专属 |
| 药师如来、十二大愿 | 卷四 · 药师 | 药师专属 |
| 曼荼罗、五智、三昧耶 | 卷五 · 密教 | 密教专属 |
| 弥勒兜率、下生 | 卷十 · 弥勒 | 弥勒专属 |
| 半跏思惟像 | 卷十 · 弥勒 | 弥勒专属 |
| 密教五明王词 | 卷五 · 密教 | 密教专属 |
| 明王 | 卷五 · 密教 | 密教专属 |

**关键区分**：
- 地藏没有"药钵/药壶/十二神将" → 可直接排除卷九
- 冥府词（阎罗、判官）≠ 密教明王
- 锡杖 + 宝珠 → 必为地藏（卷九）

### K4：权重与信任规范（卷九专版）

| 权重（W） | 信任度（T） | 含义 |
|---------|-------------|------|
| 90 | 100% | 绝对强因（地狱未空誓不成佛） |
| 85 | 95% | 强决定因（十王、泥犁、地藏核心愿） |
| 80 | 90–95% | 强因缘（地藏誓愿、阿鼻地狱、南无地藏王菩萨） |
| 75 | 80–90% | 中强因缘（拔苦、荐亡、度亡、冥府词） |
| 70 | 70–85% | 强因缘（超度、地藏名号、亡灵词） |
| 65 | 60–75% | 中因缘（罪报、历代宗亲、地狱词） |
| 60 | 50–70% | 中因缘（冥司、受罪） |
| W ≥ 75 | 推荐判定 | 可判定地藏体系 |

**判定阈值**：
- S ≥ 120 → 强地藏判定
- S ≥ 100 → 地藏倾向
- S < 100 → 弱地藏倾向，需检查语境

## 验证规则

### 判定检查清单
- [ ] 出现地藏名号（地藏菩萨、地藏王等）
- [ ] 出现六道词（地狱、饿鬼、畜生等）
- [ ] 出现冥府词（酆都、阎罗、十王等）
- [ ] 出现救度词（拔苦、超度、荐亡等）
- [ ] 无净土、密教、药师、弥勒等专属词汇
- [ ] 权重总和S ≥ 100

### 跨卷冲突处理
- **与卷三（净土）**：出现"六道 + 往生" → 分割判定；"施食 + 六道" → 卷九；"施食 + 往生句" → 卷三
- **与卷五（密教）**：冥府词绝不去卷五密教（如"冥王"≠明王）；若出现地藏真言或密教化十王像，则并列"地藏 + 密教化"
- **与卷四（药师）**：地藏没有"药钵/药壶/十二神将" → 可直接排除卷九
- **与卷十（弥勒）**：半跏思惟像 → 弥勒，非地藏

### 常见错误
- ❌ 错误：将"拔苦、荐亡"自动归入地藏（必须有地藏/地狱语境）
- ❌ 错误：将"阎罗、判官"误判为密教明王
- ❌ 错误：忽略"十王"与地藏的关联性
- ❌ 错误：将"地狱"误判为净土的"国无地狱"

### 正确示例
```
✅ 正确：
题记："南无地藏王菩萨，地狱未空誓不成佛，度亡荐拔"
主体判定：地藏菩萨（卷九）
宗派判定：南无地藏王菩萨（W=80）+ 地狱未空誓不成佛（W=90）+ 度亡（W=75）+ 荐拔（W=75）→ 地藏体系（卷九）

✅ 正确：
题记："十王审判，阎罗王，拔苦救度"
主体判定：地藏菩萨（卷九，隐含）
宗派判定：十王（W=85）+ 阎罗王（W=80）+ 拔苦（W=75）+ 救度（W=70）→ 地藏体系（卷九）

✅ 正确：
题记："泥犁地狱，阿鼻地狱，超度亡灵"
主体判定：地藏菩萨（卷九，隐含）
宗派判定：泥犁（W=85）+ 阿鼻地狱（W=80）+ 超度（W=70）+ 亡灵（W=70）→ 地藏体系（卷九）

❌ 错误：
题记："阿弥陀佛，往生极乐"
主体判定：阿弥陀佛
宗派判定：？？？
（错误：阿弥陀佛、往生极乐是净土专属，不是地藏）

❌ 错误：
题记："阎罗王，明王"
主体判定：无明确主体
宗派判定：？？？
（错误：阎罗王是地藏体系，明王是密教体系，不能混用）
```

- 卷九用于地藏菩萨主体与六道/救苦/冥府信仰的判定

- 必须有地藏/地狱语境才能判定为地藏
- 冥府词（阎罗、判官）≠ 密教明王
- 地藏没有"药钵/药壶/十二神将"
- 锡杖 + 宝珠 → 必为地藏（卷九）
- 十王出现 ≠ 必然地藏 → 必须与地藏名号/地狱词共现
