## 概览
本文件规定宗派判定的标准化输出格式，确保所有判定结果结构一致、信息完整、可追溯。

## 核心内容

### 标准输出结构

#### 【1】主体
**格式要求**：
- 明确标识造像主体
- 必须标注所属卷号

**示例**：
```
主体：地藏菩萨（卷九）
主体：弥勒（卷十）
主体：观音菩萨（卷二）
主体：药师琉璃光如来（卷四）
主体：毗卢遮那佛（卷五）
主体：阿弥陀佛（卷三）
```

**注意**：
- 主体不明 → 不得判宗派
- 主体必须明确，不能模糊表述

#### 【2】宗派（若存在）
**格式要求**：
- 明确标识宗派归属
- 必须标注所属卷号
- 主体不明时，宗派栏填"宗派未定"

**示例**：
```
宗派：天台（卷二）
宗派：净土（卷三）
宗派：密教（卷五）
宗派：华严（卷六）
宗派：禅宗（卷七）
宗派：药师信仰体系（卷四）
宗派：地藏信仰体系（卷九）
宗派：弥勒信仰体系（卷十）
宗派：民间佛教/杂修
宗派：通用大乘/宗派未定
```

**硬性规则**：
- 造像主体（观音/地藏/药师/弥勒/本师释迦等）永远不能单独作为"宗派名"出现在"判定宗派"一栏
- 主体信息应记于：线索（主体）、备注（图像体系）、或"通用大乘体系（××主体）"这种组合说明
- 真正的"宗派"栏只能填写：天台、净土、密教、华严、禅宗、三论、唯识、药师信仰体系、地藏信仰体系、弥勒信仰体系、民间佛教/杂修、通用大乘/宗派未定等

#### 【3】证据权重与信任得分
**格式要求**：
- 表格形式展示所有证据
- 每条证据包含：证据内容、所属卷号、权重W
- 最后汇总总信任得分S

**表格格式**：
```
证据 | 卷 | W
------|-----|----
"观世音" | 卷二 | 50
"普门品" | 卷二 | 85
"施无畏印" | 卷二 | 40
"往生极乐" | 卷三 | 70

总计：S = 245（成立）
```

**证据类型**：
- 造像主体名号（如观世音、药师、弥勒等）
- 经名/论名/品名（如《法华经·普门品》《金刚经》等）
- 固定术语/仪轨（如灌顶、羯磨、受戒等）
- 图像组合因（如施无畏印+净瓶、西方三圣等）
- 愿文句式（如"往生极乐"、"救苦救难"等）

#### 【4】最终判定
**格式要求**：
- 总结主体和宗派
- 说明判定依据
- 明确信任得分S
- 处理跨卷冲突说明（如有）

**格式模板**：
```
根据S=[总分]，且无跨卷冲突，判定为：[主体]（主体），[宗派]（宗派）。
```

**示例**：
```
根据S=175，且无跨卷冲突，判定为：观音（主体），天台系（宗派）。
根据S=245，存在净土+观音协同，判定为：西方三圣（主体），净土（宗派）+天台系观音（细分）。
根据S=320，密教强因W≥60，判定为：毗卢遮那佛（主体），密教（宗派）。
根据S=180，药师+净土混合，判定为：药师琉璃光如来（主体），药师净土混合信仰（宗派）。
```

### 权重与信任得分

#### 权重分级标准
| 权重范围 | 证据强度 | 说明 | 宗派决定力 |
|---------|---------|------|-----------|
| 0-25 | 弱因/中性因 | 语言背景、通用词 | 无 |
| 25-45 | 中等因 | 常见术语、一般性用语 | 弱 |
| 45-60 | 中强因 | 主体名号、经名、固定术语 | 中 |
| 60-80 | 强因/决定因 | 宗派核心词、专有术语 | 强 |
| 80-100 | 绝对因 | 不可争议的证据 | 决定性 |

#### 信任得分S计算规则
- 单卷得分 = 该卷所有证据权重之和
- 总得分S = 所有卷得分之和
- S ≥ 100：判定成立
- S < 100：判定不成立，需要更多证据

### 计算模板

#### 📐 模板A：单条处理
**用途**：处理单个判定词

**步骤**：
1. 输入：判定词X
2. 查找：卷号V、权重W
3. 换算：信任度S与权重W
4. 输出：{X, V, W}

**示例**：
```
输入：判定词"观世音"
查找：卷号V=卷二，权重W=50
换算：S=50
输出：{"观世音", "卷二", 50}
```

#### 📐 模板B：整段铭文处理
**用途**：处理完整题记文本

**步骤**：
1. 抽取全部判定词 → {Xi}
2. 为每个Xi查找：卷号Vi，权重Wi
3. 按卷分组求和
4. 主体卷优先筛选（卷2/3/4/9/10）
5. 宗派卷次级筛选（卷5/6/7/11/12/13/14/15）
6. 最大S的卷为最终判定卷

**示例**：
```
题记："观世音菩萨，普门品，愿往生西方极乐世界"
Step 1：判定词 = {"观世音", "普门品", "往生极乐", "西方"}
Step 2：
- "观世音" → 卷二，W=50
- "普门品" → 卷二，W=85
- "往生极乐" → 卷三，W=70
- "西方" → 卷三，W=40
Step 3：
- 卷二得分：50+85=135
- 卷三得分：70+40=110
- 总得分：S=245
Step 4：主体卷优先 → 卷二（观音）、卷三（净土）
Step 5：宗派卷判定 → 天台系（卷二）、净土（卷三）
Step 6：最终判定 → 观音（主体），净土（宗派）+天台系观音（细分）
```

#### 📐 模板C：交叉冲突处理
**用途**：处理跨卷冲突

**步骤**：
1. 若出现密教强因W≥60 → 直接进入卷五
2. 若主体卷冲突（如地藏 + 弥勒）→ 依据主体优先级选定主体卷
3. 卷间互斥矩阵检查 → 若两卷冲突 → 保留主体卷

**示例**：
```
题记："造药师如来，愿往生东方琉璃世界"
Step 1：检查密教强因 → 无
Step 2：主体卷冲突 → 卷四（药师）vs 卷三（净土）
       优先级：卷四（药师）> 卷三（净土）
       选择：卷四
Step 3：互斥矩阵检查 → ⛔ 绝对排除，保留主体卷（卷四）
Step 4：最终判定 → 药师琉璃光如来（主体），药师信仰体系（宗派）
```

#### 📐 模板D：最终输出格式
**用途**：生成标准化输出

**格式模板**：
```
主体：X（卷号）
宗派：Y（卷号）
主要证据：
- 词1（卷号，W）
- 词2（卷号，W）
信任得分：S = [总分]
最终结果：
依据主体判定 + 卷间互斥 + 信任得分，判定为……
```

## 验证规则

### 输出检查清单
- [ ] 包含【1】主体，明确标注卷号
- [ ] 包含【2】宗派，明确标注卷号
- [ ] 包含【3】证据权重表格，格式正确
- [ ] 包含【4】最终判定，说明依据
- [ ] 主体不能作为宗派名
- [ ] 信任得分S ≥ 100才判定成立
- [ ] 跨卷冲突已处理说明

### 常见错误
- ❌ 错误：主体不明确时仍判宗派
- ❌ 错误：主体（观音）作为宗派名输出
- ❌ 错误：证据权重表格格式错误
- ❌ 错误：信任得分S < 100仍判定成立
- ❌ 错误：未处理跨卷冲突
- ❌ 错误：密教强因未绝对优先

### 正确示例
```
✅ 正确：
主体：观世音菩萨（卷二）
宗派：天台（卷二）
证据权重表格完整
信任得分S=175 ≥ 100
最终判定明确

❌ 错误：
主体：观世音菩萨
宗派：观世音（错误：主体作为宗派）
证据权重不完整
信任得分S=50 < 100仍判定成立
```

## 约束与注意事项
- 必须严格按照标准化格式输出
- 主体不能作为宗派名
- 信任得分S ≥ 100才判定成立
- 密教强因（W≥60）绝对优先
- 跨卷冲突必须明确说明处理方式
- 证据链必须完整透明
