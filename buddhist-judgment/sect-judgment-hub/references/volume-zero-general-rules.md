## 概览
卷零是宗派判定十五卷系统的母规则层，定义了全局框架、卷间关系、权重体系、交叉逻辑和输出格式。所有判定必须先遵守卷零总则，然后再应用各卷具体规则。

## 核心内容

### 卷间总则摘要

#### #0.1 通用硬锚点（卷1–15共用）
硬锚点类型包括：
- 造像主体名号（如观世音、药师、弥勒、地藏等）
- 经名/论名/品名（如《法华经·普门品》《金刚经》等）
- 固定术语/仪轨（如灌顶、羯磨、受戒等）
- 图像组合因（如施无畏印+净瓶、西方三圣等）
- 愿文句式（如"往生极乐"、"救苦救难"等）

#### #0.2 证据链透明性要求
每条判定词必须具备：
- 【出处】字段：经论章疏题名、可定位的典籍条目、或题记原刻题名
- 【权重W】：0–100，反映证据强度
- 【卷号归属V】：明确归属于哪一卷
- 【判定逻辑】：说明如何从证据推导出结论

高泛化词（如空、识、戒、观心、圆、实相等）必须写明【限定共现】（与哪些硬锚点同现才计分/才升权），否则默认降为弱因缘，且不得触发Gate。

#### #0.3 Gate规则（阈值控制）
- W ≥ 60：强因，触发宗派决定，直接进入对应卷
- 45 ≤ W < 60：中强因，可作为重要证据，需组合判断
- 25 ≤ W < 45：中等因，作为佐证
- W < 25：弱因，仅作为语言背景

特殊规则：
- 密教（卷五）：W ≥ 60 直接入卷五，绝对优先
- 卷一（中性语料层）：W ≤ 25，不触发Gate

#### #0.4 权重分级标准
| 权重范围 | 证据强度 | 说明 | 宗派决定力 |
|---------|---------|------|-----------|
| 0-25 | 弱因/中性因 | 语言背景、通用词 | 无 |
| 25-45 | 中等因 | 常见术语、一般性用语 | 弱 |
| 45-60 | 中强因 | 主体名号、经名、固定术语 | 中 |
| 60-80 | 强因/决定因 | 宗派核心词、专有术语 | 强 |
| 80-100 | 绝对因 | 不可争议的证据 | 决定性 |

### 各卷总则详解

#### 卷一总则（经文词干库·中性）
**适用范围**
收录各大经典（法华、华严、般若、心经、维摩、楞严、涅槃等）的中性经文词干、品名、通用术语。

**功能定位**
仅作为语言背景与经文指向的基础库，不直接判定主体、不直接决定宗派。

**与主体/宗派的关系**
- 卷一的词条不能单独用来认定造像主体
- 卷一的词条不能单独用来判定宗派
- 只能作为其他卷（卷二–卷十二）中"宗派词"的语义来源和出处说明

**权重使用原则**
- 卷一中所有词条的基础权重通常 ≤ 25
- 属于"弱因"或"中性因"
- 只有当卷一词条与其他卷中的"宗派决定因/强因缘"词条同时出现时，才可以作为加分的语言佐证

**交叉引证方式**
- 当卷二–卷十二中的词条需要指明"出自某经某品"，统一指向卷一中对应条目
- 卷一负责记载：经号、卷次、原文关键句，用于后续学术引用与注脚

**负向因与使用禁则**
- 不得将卷一的任何"经名"或"品名"本身，当作宗派标签使用
- 禁止："有法华两字就判天台"
- 不得从卷一词条反推出现代宗派标签
- 禁止："心性""般若"直接判为禅宗/唯识

**特别说明**
卷一是十二卷的基础语料层，负责保证"每条判定词都有经典可查"，但本身不参与得分决策，只提供出处与语境。

#### 卷二总则（观音体系·天台/心经/密教/普门）
**适用范围**
专门收录观音主体相关的题记用语、经文名号、图像组合线索。

**功能定位**
只在主体已被确定为观音的前提下启用本卷的宗派判定。

**主体与宗派关系**
主体"观音"必须先通过：
- 铭文名号（观世音、观自在、圣观音等）
- 或图像组合因（施无畏印 + 净瓶 + 救苦愿文等）
判定成功后，本卷才参与"天台系观音/心经系观音/密教观音/民间观音"的细分。

**权重使用原则**
- 与《法华经·普门品》高度对应的词（海潮音、普门示现等）→ 视为宗派强因缘（45–55）
- 与心经观自在体系对应的词 → 多为中强因缘（35–45）
- 密教观音（如如意轮、千手、准提、马头等）→ 在主体已确认为观音后，可给到45–55的密教倾向

**交叉引证方式**
- 与卷一：说明这些观音名号出自何经、何品
- 与卷三：当观音与阿弥陀/西方净土同时出现时，交叉卷三以判定"净土 + 观音协同体系"
- 与卷五：出现"观音 + 明王/曼荼罗结构/真言"时，通过卷五确认是否进入"观音密教化"通道

**负向因与使用禁则**
- 禁止仅凭"慈悲、大悲、大士"这类弱因名号判定为观音或天台宗
- 当题记中明确指向地藏、药师、弥勒、阿弥陀时，卷二应后退，仅作语言背景，不可争夺主体

**特别说明**
卷二不负责判定"是否观音"，只在主体已确认后，进行观音内部宗派与流派细分。

#### 卷三总则（净土体系·阿弥陀/极乐世界）
**适用范围**
以《阿弥陀经》《无量寿经》《观无量寿经》三经词干为核心，覆盖：
- 极乐世界名号
- 往生愿文
- 九品往生用语
- 西方三圣图像结构等

**功能定位**
判定净土信仰的宗派归属。

**主体与宗派关系**
- 主体可为阿弥陀佛
- 可为"西方三圣"（阿弥陀 + 观音 + 势至）
- 甚至是"净土变相"
- 只要题记明确出现"往生、极乐、九品、莲华化生"等净土核心词，即视为净土体系的强因缘

**权重使用原则**
- "三经直引 + 西方极乐 + 九品"组合 → 宗派决定因（70–80），直接判为净土信仰
- 仅有"阿弥陀"名号，而无任何净土语境时 → 权重下调至40–50，需要看与卷二、卷五等的组合
- "荐亡、拔苦"若明确与"往生极乐"相连 → 权重可提升为净土 + 地藏协同

**交叉引证方式**
- 与卷二：西方三圣中观音的角色由卷二细分，不改变净土大框架
- 与卷四：严禁把"药师"主体误判为净土，卷四的"十二大愿/东方琉璃世界"若出现，即优先归药师
- 与卷九：若出现"地藏 + 往生 + 度亡"的组合，则视为净土 + 地藏交织，须两卷并列标记

**负向因与使用禁则**
- 禁止因单一"莲花、莲座、宝池"等图像要素而判定净土
- 若铭文明确为"东方琉璃世界/药师十二大愿"，卷三必须退出，不得争夺宗派

**特别说明**
卷三强调的是"往生极乐"的宗派向度，而非一般"阿弥陀像"本身的存在。

#### 卷四总则（药师体系·十二大愿/药师七佛）
**适用范围**
专门处理"药师琉璃光如来"及其十二大愿、十二神将、药师七佛等信仰体系。四川地区独立的药师信仰（非单纯附属净土）须在本卷中独立建模。

**功能定位**
判定药师信仰体系的宗派归属。

**主体与宗派关系**
- 主体为药师佛/药师七佛/药师净土变相时，宗派归属优先视为药师体系，而非净土
- 当题记出现"十二大愿、东方琉璃世界、十二神将"等关键字，即直接确立药师主体

**权重使用原则**
- "药师 + 十二大愿/琉璃世界/十二神将" → 宗派决定因（70–80），优先级高于净土
- "药师 + 药叉神将" → 强因缘（45–55），基础上可由图像组合补加
- "单独药师佛名号" → 权重控制在35–45，不强行判定宗派，需结合愿文/神将和其他卷

**交叉引证方式**
- 与卷三：如题记同时出现"往生极乐"语汇，本卷与卷三需并列 → "药师净土混合信仰"
- 与卷五：出现真言、陀罗尼幢、"佛顶尊胜陀罗尼"等密教要素，则与卷五交叉为"药师 + 密教化"

**负向因与使用禁则**
- 禁止把所有"有药师"题记一律归入净土（这是本工程的核心负面规则）
- 禁止用"无量寿、无量光"等纯属阿弥陀专用名号来支持药师判定

**特别说明**
卷四是独立的药师信仰体系，不是净土的子集。

（由于篇幅限制，卷五至卷十五的总则详见后续集成版本）

### AI输出格式（必须遵守）

#### 标准输出结构
AI必须以如下结构输出判断：

**【1】主体**
例如：
- "主体：地藏菩萨（卷九）"
- "主体：弥勒（卷十）"

**【2】宗派（若存在）**
例如：
- "宗派：天台（卷二）"
- "宗派：净土（卷三）"

注意：主体不明 → 不得判宗派。

**【3】证据权重与信任得分**
格式：
| 证据 | 卷 | W |
|------|-----|---|
| "观世音" | 卷二 | 50 |
| "普门品" | 卷二 | 85 |
| "施无畏印" | 卷二 | 40 |

总计：S = 175（成立）

**【4】最终判定**
**硬性规则**：造像主体（观音/地藏/药师/弥勒/本师释迦等）永远不能单独作为"宗派名"出现在"判定宗派"一栏。

其信息应记于：线索（主体）、备注（图像体系）、或"通用大乘体系（××主体）"这种组合说明。

真正的"宗派"栏只能填写：
- 天台、净土、密教、华严、禅宗、三论、唯识
- 药师信仰体系、地藏信仰体系、弥勒信仰体系
- 民间佛教/杂修、通用大乘/宗派未定等

例如：
"根据S=175，且无跨卷冲突，判定为：观音（主体），天台系（宗派）。"

#### 权重→信任得分计算模板（AI使用）

**📐 模板A：单条处理**
- 输入：判定词X
- 查找：卷号V、权重W
- 换算：信任度S与权重W
- 输出：{X, V, W}

**📐 模板B：整段铭文处理**
- Step 1：抽取全部判定词 → {Xi}
- Step 2：为每个Xi查找：卷号Vi，权重Wi
- Step 3：按卷分组求和
- Step 4：主体卷优先筛选（卷2/3/4/9/10）
- Step 5：宗派卷次级筛选（卷5/6/7/11/12/13/14/15）
- Step 6：最大S的卷为最终判定卷

**📐 模板C：交叉冲突处理**
- Step 1：若出现密教强因W≥60 → 直接进入卷五
- Step 2：若主体卷冲突（如地藏 + 弥勒）→ 依据主体优先级选定主体卷
- Step 3：卷间互斥矩阵检查 → 若两卷冲突 → 保留主体卷

**📐 模板D：最终输出格式**
```
主体：X（卷号）
宗派：Y（卷号）
主要证据：
- 词1（卷号，W）
- 词2（卷号，W）
信任得分：
最终结果：
依据主体判定 + 卷间互斥 + 信任得分，判定为……
```

## 验证规则

### 判断流程
1. 优先读取并理解卷零总则
2. 应用通用硬锚点规则
3. 按卷序执行判定（卷一→卷十五）
4. 计算证据权重与信任得分
5. 应用卷间交叉逻辑处理冲突
6. 输出标准化结果

### 边界情况处理
- 卷一词条不能单独判定宗派
- 密教强因（W≥60）绝对优先
- 主体卷优先于宗派卷
- 跨卷冲突按互斥矩阵处理
- 高泛化词必须明确限定共现条件

## 约束与注意事项
- 卷零总则是母规则，所有判定必须遵守
- 不得跳过卷零直接应用具体卷规则
- 输出必须严格按照标准化格式
- 证据链必须透明，包含出处、权重、卷号
- 不得将造像主体单独作为宗派名输出
- 跨卷冲突必须按互斥矩阵处理
